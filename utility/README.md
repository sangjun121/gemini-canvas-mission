## 앱 이름: 꼬박꼬박
## 카테고리: 유틸리티 앱

### 배포 링크
[꼬박꼬박 바로가기](https://gemini.google.com/share/4a013da9dc39)

### 이 앱을 만든 이유
- 어떤 문제/불편함을 해결하려고 했나요?
> 영양제 및 물을 금일 섭취했는지 한 서비스 내에서 한번에 확인하기 어려워, 한 플랫폼에서 확인할 수 있도록 개발하였습니다.  

- 이 앱을 언제, 어떻게 사용할 건가요?
> 매일 아침/점심/저녁 지정한 영양제를 섭취 후에 체크하는 용도로 사용할 예정입니다. 또한, 당일 물 섭취량을 지속적으로 추적할 때 사용할 예정입니다.

### 주요 기능
#### 최소 기능 요구 사항 선 정의

- 아침/점심/저녁(이하, 카테고리) 단위로 1회 영양제 등록/수정/삭제 기능
- 영양제 단위로 1회 복용량 등록/수정/삭제 기능
- 하루 목표 물 섭취량 등록/수정 기능
- 물 섭취 후 섭취량 등록 기능
- 캘린더로 이전 날짜 섭취 기록 조회 기능
