## 앱 이름: 우아한 퍼즐
## 카테고리: 게임

### 배포 링크
[우아한 퍼즐 바로가기](https://gemini.google.com/share/8b10a4481c31)

### 이 앱을 만든 이유
- 어떤 문제/불편함을 해결하려고 했나요?
> 어렸을 때 자주했던 추억의 게임인 사천성을 구현하여, 우테코 진행 간에 스트레스 해소를 위해 구현하였습니다!

- 이 앱을 언제, 어떻게 사용할 건가요?
> 쉬는 시간 잠시 스트레스 해소를 위해 사용 예정입니다.

### 주요 기능
#### 최소 기능 요구 사항 선 정의

- 초기 설정(타일 생성 및 화면 배치): 사용자가 짝수개(2~50사이의) 짝수개의 타일 개수를 선택한다. 보드에 같은 종류의 타일들이 짝수개로 배치된다. 두개의 같은 타일이 선택되면 연결 가능해. 연결 가능 조건은 아래 추가할 예정이야.
- 게임 진행 조건: 사용자가 제거 가능한 타일 두개를 선정하면 해당 타일 두개가 제거된다.
- 게임 종료조건: 모든 타일이 사라지면 게임이 종료된다.
- 연결 가능조건: 최대 두 번 꺾이는 경로, 중간에 다른 타일 없음)에 맞게 연결 가능하면 두 타일을 제거한다.
